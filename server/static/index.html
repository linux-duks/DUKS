DUKS

Dashboard for Unified Kernel Statistics

An analytical platform providing insights into Linux Kernel development cycles, contributor patterns, and collaborative dynamics within one of the world's largest open-source projects.

Explore Dashboard Learn More

About DUKS

DUKS provides comprehensive analytics and visualizations of the Linux Kernel development process, helping researchers and developers understand collaboration patterns, contribution trends, and project dynamics.

Contributor Analysis

Track unique contributors, maintainers, reviewers, and their collaboration patterns over time.

Code Metrics

Analyze lines of code changes, commit patterns, and development velocity across kernel versions.

Interactive Visualizations

Explore data through customizable charts with multiple time windows and filtering options.

Research & Publications

DUKS is part of ongoing research into open-source collaboration patterns and software engineering practices in large-scale projects.
